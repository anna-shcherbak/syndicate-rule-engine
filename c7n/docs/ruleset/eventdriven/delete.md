# delete

## Description

Deletes Event-driven ruleset

## Synopsis

```bash
c7n ruleset eventdriven delete
    --name <text>
    [--version <float>]
```

## Options

`--name` (text) 

Ruleset name

`--version` (float) [default: 1.0]

Event-driven ruleset version to delete
