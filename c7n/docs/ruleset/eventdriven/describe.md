# describe

## Description

Describes Event-driven ruleset

## Synopsis

```bash
c7n ruleset eventdriven describe
    [--cloud <AWS|AZURE|GCP>]
    [--get_rules <boolean>]
```

## Options

`--cloud` (AWS, AZURE, GCP) 

Event-driven ruleset cloud to describe

`--get_rules` (boolean) 

If specified, ruleset's rules ids will be returned. MAKE SURE to use '--json' flag to get a clear output
