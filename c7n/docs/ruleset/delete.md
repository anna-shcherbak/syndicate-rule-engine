# delete

## Description

Deletes Customer ruleset. For successful deletion, the ruleset must be
    inactive

## Synopsis

```bash
c7n ruleset delete
    --name <text>
    --version <float>
```

## Options

`--name` (text) 

Ruleset name

`--version` (float) 

Ruleset version
