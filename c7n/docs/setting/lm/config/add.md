# add

## Description

Adds License Manager access configuration data

## Synopsis

```bash
c7n setting lm config add
    --host <text>
    --port <integer>
    --version <float>
```

## Options

`--host` (text) 

License Manager host.

`--port` (integer) 

License Manager port.

`--version` (float) [default: 2.0]

License Manager version.
