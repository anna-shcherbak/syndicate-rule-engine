# update

## Description

Updates settings of your user's tenant

## Synopsis

```bash
c7n tenant update
    [--rules_to_exclude <text>]
    [--rules_to_include <text>]
    [--tenant_name <text>]
```

## Options

`--rules_to_exclude` (text) 

Rules to exclude for tenant

`--rules_to_include` (text) 

Rules to include for tenant

`--tenant_name` (text) 

Name of related tenant
