# delete

## Description

Deletes Custodian Service Credentials Manager configuration.

## Synopsis

```bash
c7n tenant credentials delete
    --account_number <text>
    --cloud <text>
```

## Options

`--account_number` (text) 

Cloud native account identifier

`--cloud` (text) 

The cloud to which the credentials configuration belongs.
