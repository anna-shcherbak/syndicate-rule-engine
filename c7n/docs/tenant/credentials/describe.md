# describe

## Description

Describes Custodian Service Credentials Manager configurations.

## Synopsis

```bash
c7n tenant credentials describe
    [--account_number <text>]
    [--cloud <text>]
```

## Options

`--account_number` (text) 

Cloud native account identifier

`--cloud` (text) 

The cloud to which the credentials configuration belongs.
