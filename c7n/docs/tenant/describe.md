# describe

## Description

Describes your user's tenant(s)

## Synopsis

```bash
c7n tenant describe
    [--account_number <text>]
    [--full <boolean>]
    [--limit <integer range>]
    [--next_token <text>]
    [--tenant_name <text>]
```

## Options

`--account_number` (text) 

Cloud native account identifier

`--full` (boolean) 

Show full command output.

`--limit` (integer range) [default: 10]

Number of records to show

`--next_token` (text) 

Token to start record-pagination from

`--tenant_name` (text) 

Name of related tenant
