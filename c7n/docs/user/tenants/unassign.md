# unassign

## Description

Detaches tenants from a user

## Synopsis

```bash
c7n user tenants unassign
    --username <text>
    [--all_tenants <boolean>]
    [--tenant <text>]
```

## Options

`--username` (text) 

User to update

`--all_tenants` (boolean) 

Remove all tenants from user. This will allow the user to interact with all tenants within the customer.

`--tenant` (text) 

Tenant names to unassign from user
