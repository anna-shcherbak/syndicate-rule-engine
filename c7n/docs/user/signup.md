# signup

## Description

Signs up a new Cognito user

## Synopsis

```bash
c7n user signup
    --role <text>
    --username <text>
    [--password <text>]
    [--tenants <text>]
```

## Options

`--role` (text) 

User`s role

`--username` (text) 

User`s username. Must be unique

`--password` (text) 

User`s password

`--tenants` (text) 

Tenants within the customer the user need access to
