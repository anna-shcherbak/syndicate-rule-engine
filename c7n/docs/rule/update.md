# update

## Description

Pulls the latest versions of rules within your customer

## Synopsis

```bash
c7n rule update
    [--all <boolean>]
    [--git_project_id <text>]
```

## Options

`--all` (boolean) 

Set to update rules in all the customers. Only for the SYSTEM customer

`--git_project_id` (text) 

Git repo project id to pull update from
