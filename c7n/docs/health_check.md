# health_check

## Description

Checks Custodian Service components availability

## Synopsis

```bash
c7n health_check
    [--identifier <text>]
    [--status <OK|UNKNOWN|NOT_OK>]
```

## Options

`--identifier` (text) 

Concrete check id to retrieve

`--status` (OK, UNKNOWN, NOT_OK) 

Filter checks by status
