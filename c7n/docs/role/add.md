# add

## Description

Creates the Role entity with the given name from Customer with the given id

## Synopsis

```bash
c7n role add
    --name <text>
    --policies <text>
    [--expiration <text>]
```

## Options

`--name` (text) 

Role name

`--policies` (text) 

List of policies to attach to the role

`--expiration` (text) 

Expiration date, ISO 8601. Example: 2021-08-01T15:30:00
