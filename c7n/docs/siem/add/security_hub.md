# security_hub

## Description

Adds security hub configuration.

## Synopsis

```bash
c7n siem add security_hub
    --product_arn <text>
    --region <text>
    [--tenant_name <text>]
    [--trusted_role_arn <text>]
```

## Options

`--product_arn` (text) 

ARN of security product

`--region` (text) 

AWS region name

`--tenant_name` (text) 

Name of related tenant

`--trusted_role_arn` (text) 

Role that will be assumed to upload findings
