# describe

## Description

Describes Custodian Service Scans

## Synopsis

```bash
c7n job describe
    [--job_id <text>]
    [--limit <integer range>]
    [--next_token <text>]
    [--tenant_name <text>]
```

## Options

`--job_id` (text) 

Job id to describe

`--limit` (integer range) [default: 10]

Number of records to show

`--next_token` (text) 

Token to start record-pagination from

`--tenant_name` (text) 

Name of related tenant
